# REPORTS-3438_VAS_tracking
VAS tracking and revenue attribution for OneAssist
This project tracks Value Added Services (VAS) across all OneAssist product categories — PE, HA, Finance, Motor, and Furniture — and attributes plan revenue to individual services based on their COGS weight.
The VAS tracking output directly feeds into the Cross-Sell Awareness Scoring Model (vas_redeemed parameter, weight = 3) under sibling project REPORTS-3439.
